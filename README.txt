alu-16bit_CS4341_Fall2019
This is the semester team project for Digital Logic. This project is a 16 bit ALU coded in iverilog.

Upload the all files to a folder. Change directory to that folder. Compile and run testbenchALU.v on the command line by performing the following sequence of statements:
$ - iverilog testbenchALU.v
$ - vvp a.out
